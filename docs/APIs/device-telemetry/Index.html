    Device Telemetry

    Device Telemetry service is used to manage Alarms, Rules, AlarmsByRules, Message and Device Files.

    Within each Serenity instance an interactive Swagger UI is available at the following path /auth/index.html for more information, click here

    Rest Operation Groups

    The Device Telemetry REST API provides operations for working with the following:

    Operation group Description
    Alarms Manage Alarms
    AlarmsByRule Fetch paginate details of Rules with Alarms count and Alarms for specific rule
    DeviceFiles Fetch device files detail and download device file
    Messages Manage Messages
    Rules Manage Rules
    Status Health status operations for the Service
